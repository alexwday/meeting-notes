# Meeting Notes Transcriber

## Mac

### Clone and run

```bash
git clone <repo-url>
cd meeting-notes
python3 launch.py
```

### If the folder name is different

```bash
cd meeting-notes-main
python3 launch.py
```

### If `python3` is not the right command

```bash
python launch.py
```

### If the browser does not open automatically

```bash
open http://127.0.0.1:8765
```

## Windows

### Clone and run

```powershell
git clone <repo-url>
cd meeting-notes
py launch.py
```

### If the folder name is different

```powershell
cd meeting-notes-main
py launch.py
```

### If `py` is not the right command

```powershell
python launch.py
```

### If the browser does not open automatically

```powershell
start http://127.0.0.1:8765
```
